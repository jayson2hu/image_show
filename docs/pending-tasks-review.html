卡片交互规范

  • 默认态 — 白底 + 1px 边框(slate-200) + 12px 圆角 + 微阴影(0 2px 8px rgba(0,0,0,.04))
  • Hover — border → violet-300，scale(1.02)，阴影加深，图标 translateY(-2px) 微跳
  • 选中态 — 2px border violet-500 + 背景 violet-50 + 外发光 box-shadow 0 0 0 3px rgba(139,92,246,.15) + 左上角小勾标记
  • 点击反馈 — 0.15s scale(0.97→1) 弹性回弹
  • 再次点击已选中卡片 — 取消选中，清空提示词，恢复默认比例(1:1)
  • transition — all 0.2s cubic-bezier(0.4, 0, 0.2, 1)

提示词填充动画

  • 点击场景 → 输入框聚焦 → 模板文字以打字机效果逐字出现（40ms/字 + 光标闪烁）
  • 打字完成 → 光标移到末尾，用户可继续编辑
  • 输入框已有内容 → 先 0.15s fade out 清空，再填入新内容
  • 「自由创作」→ 不填充提示词，仅切换比例(1:1)，输入框获焦 + placeholder "描述你想要的图片..."

响应式布局

  • ≥1024px 桌面 — 3 列网格，gap 16px
  • 768-1023px 平板 — 2 列网格，gap 12px
  • <768px 手机 — 横向滚动卡片（scroll-snap-type: x mandatory），每次显示 2.3 张，露出第三张边缘引导滑动
