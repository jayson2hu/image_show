# 大功能 B 套餐与人工充值验收记录

日期：2026-05-08

## 范围

- B1 套餐模型和默认套餐核查。
- B2 后台套餐管理 UI 优化。
- B3 人工充值联系方式配置。
- B4 前台购买中心展示人工充值联系方式。

## 验收结果

- 新部署没有套餐时，可初始化默认积分套餐。
- 后台套餐页支持新增、编辑、启停、删除和排序展示。
- 后台设置页支持独立配置人工充值联系方式，不复用图片生成渠道。
- 前台购买中心展示套餐和人工充值联系信息。
- 当前版本不创建真实支付订单，不跳转第三方支付，避免用户误解为自动支付。
- 公开联系接口只返回用户可见的联系方式，不暴露 WeChat Server Token、R2 Secret、渠道密钥等敏感配置。
